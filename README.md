# fluffy-Krypto-Guide
Simulering av Kryptering ved binær koding

Genererer 5 tilfeldige markørpunkter på en 8×8 LED‑matrise
Markørene blinker hvitt/gult
Brukeren velger én av de fem markørene + krypteringsnøkkel
Koordinaten blir kryptert med XOR
XOR‑prosessen vises intuitivt og visuelt, slik du allerede har i koden din
